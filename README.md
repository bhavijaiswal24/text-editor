# Text Editor

A console-based Text Editor application developed in Java using Object-Oriented Programming (OOP) principles. The project simulates the basic functionalities of a text editor, allowing users to create, edit, and manage text efficiently.

## Features

- Create and edit text content
- Perform text manipulation operations
- Execute editor actions through dedicated modules
- Save and manage text data
- User-friendly console interface
- Modular and extensible architecture

## Technologies Used

- Java
- Object-Oriented Programming (OOP)

## Project Structure

```
text-editor/
│
├── src/
│   ├── action/         # Contains editor actions and operations
│   ├── notepad/        # Core text editor functionalities
│   ├── App.java        # Main entry point of the application
```

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/bhavijaiswal24/text-editor.git
```

2. Navigate to the project directory:

```bash
cd text-editor
```

3. Compile the Java files:

```bash
javac src/App.java
```

4. Run the application:

```bash
java App
```
